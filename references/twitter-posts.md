## Thread 1: Viral Transformation (310K Views)

**Type:** Transformation Thread
**Performance:** 310K views

### Full Thread:

**Tweet 1 (Hook):**
3 months ago, I bought a business making $500/month for $4,000.

It's now worth $100k

**Tweet 2:**
Please try this at home, but you need a lot of luck and hard work

**Tweet 3:**
The business was @sourcelyai - an AI tool for academic research

When I bought it:
- 200 users
- $500 MRR
- One solo founder who wanted out

**Tweet 4:**
The growth formula that worked:

1. Acknowledge it's small, value each user
2. Get into user's head
3. Create "Jonny" — extremely well-defined user persona
4. Pick one specific problem to solve

**Tweet 5:**
What you need is a Jonny. 🤓

Jonny is a 20-year-old college junior. He's writing a research paper on climate change. He has 47 tabs open and no idea which sources to cite.

That's who I build for. One person.

**Tweet 6:**
When everything is for Jonny, decisions get easy:
- Would Jonny pay $9/month? Yes
- Would Jonny use this feature? No
- Would Jonny share this with friends? Maybe

**Tweet 7:**
3 months later:
- 2,000+ users
- $4,000 MRR
- Worth ~$100k at market multiples

10x in 90 days.

**Tweet 8:**
The unsexy truth: Most of the work was fixing broken stuff.
- Rewrote the checkout flow
- Fixed 47 bugs
- Migrated to better hosting
- Set up proper analytics

Not glamorous. But necessary.

**Tweet 9:**
If you're thinking about buying a small business:
1. Start smaller than you think
2. Buy something you understand
3. Talk to existing users immediately
4. Create your Jonny
5. Fix the basics before adding features

**Tweet 10 (CTA):**
I write about micro acquisitions, building in public, and buying businesses on acquire.com

Follow along if you want to see the journey.

DMs are open if you have questions 🤝
