# Content Formulas for Dev's Voice

Proven structures adapted to Dev's style.

---

## Thread Structures

### The Transformation Thread
```
Hook: [Before state] → [After state]
"I bought X for $4K. 18 months later, sold for six figures."

Context: Quick backstory (2-3 tweets)

Lessons: 5-10 numbered points
Each: **Bold title** + 2-3 sentences + specific example

CTA: Question or follow prompt
```

### The Contrarian Thread
```
Hook: Challenge common belief
"Everyone says [X]. They're wrong."

Proof: Why the common belief fails (2-3 tweets)

Reality: What actually works (3-5 tweets with examples)

CTA: "What's your take?"
```

### The Breakdown Thread
```
Hook: Interesting case teaser
"Imagine raising $150K and selling for $5K 💀"

Analysis: Walk through the details
- Numbers, ratings, assessments

Takeaways: What to learn from this

CTA: Opinion question
```

### The Journey Thread
```
Hook: Timeline or milestone
"3 years ago I [past state]. Today [current state]."

Chapters: Key moments (chronological)
Each: Date/period + what happened + lesson

Reflection: Biggest learnings

CTA: "What's your journey?"
```

## Post Structures

### The Hot Take
```
[Bold statement]. [One sentence of backing]. [Question].

"Raising VC is the worst thing you can do for most businesses. 
97% of startups fail to return capital. What if you just... made money instead?"
```

### The Lesson Post
```
[Observation from experience].

Here's what I learned:

• Point 1
• Point 2
• Point 3

[Question for engagement]
```

### The Story Post
```
[Hook: What happened]
[Context: Quick setup]
[Tension: The problem/challenge]
[Resolution: What we did]
[Lesson: The takeaway]
[CTA: Question]
```

## Headline Formulas (Dev-ified)

**Transformation:**
- "I [past state]. [Timeframe] later, [outcome]."
- "[Small input] → [big output] in [time]"

**Contrarian:**
- "Most people think [X]. That's wrong."
- "Unpopular opinion: [bold claim]"
- "[Common advice] is garbage. Here's what works:"

**Curiosity:**
- "[Big number] [things]. [Small number] [results]. Here's what I learned:"
- "The real reason [outcome] (it's not what you think)"

**Social proof:**
- "[Impressive metric] in [timeframe]. Here's how:"
- "Last [time period]: [results]. Thread:"

**Challenge:**
- "Why doesn't everyone know you can [surprising thing]?"
- "Imagine [doing X]. That's literally possible."

## CTA Formulas

**Engagement:**
- "What would you do differently?"
- "What's your take?"
- "Reply with [X] if you want [Y]"
- "Agree or disagree?"

**Follow:**
- "Follow for more [topic]"
- "I write about [X] daily"

**Action:**
- "DMs are open"
- "Link in bio"
- "Drop a [emoji] if this resonates"

## Repurposing Flow

**One long-form piece → Multiple formats:**

1. **Newsletter** → Core insight (1,500 words)
2. **Twitter thread** → Key points (8-12 tweets)
3. **LinkedIn post** → Main lesson (1,000 chars)
4. **Tweet** → Best one-liner hook
5. **IG carousel** → Visual breakdown (5-8 slides)

Each format gets its own spin — never copy-paste across platforms.
